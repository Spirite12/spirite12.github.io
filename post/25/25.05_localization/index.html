系统设计

  • 本地化内容主要包括：文本本地化 与 资源本地化。
  • 文本本地化：
    • 文本本地化主要包含三类来源：1）预制体文本；2）代码文本；3）表数据文本
      • 预制体上文本、代码文本：统一在 文本表 内处理；
      • 表文本：统一在 默认表 与 枚举表 内处理；
    • 文本本地化规则遵守：TableRules
  • 资源本地化：
    • 资源本地化：用于处理不同类型资源的本地化加载。
    • 资源本地化规则由 LocalizeRules 进行统一管理。

  • 本地化模块划分：

    • 默认表、枚举表：处理所有表字段需要的本地化；
    • 文本表：处理预制体上文本、代码内的文本；
    • 常量表：不需要处理本地化；

  • 除了默认表，其他三个表都有严格遵守的表字段；

    • 具体表字段由： TableRulesTypeXXX 内查找 TableXXXClass 类 可见；
      • XXX：为枚举值的命名；

默认表 Default

作用：根据表字段是否需要本地化来生成对应的本地化表代码；

  1. 项目工程：链接

    • 克隆项目，并打开项目

  2. 创建英文本地化资源：

    • 找到工程路径：Game\Settings \Localize
      • LocalizeRules：规则工具
      • Localization Settings：本地化配置数据
      • Chinese (Simplified) (zh-CN)：中文本地化包
    • 文件夹内只有中文包，点击 Localization Settings，点击 Locale Generator；
    • 在 Locale Generactor 窗口内找到 Enghlish（en），点击生成；生成位置就在当前文件夹下；
    • 此时可以看到生成 English (en) 英文包，Localization Settings也关联了英文；

  3. 生成本地化数据：

    • 查找 Game\Table\TableTest.csv

      • 查看表内容，可知 Name 字段需要进行本地化处理；
      • 根据实际项目情况可删除，表都放在这个文件夹下；

    • 找到 Game\Settings\Table\TableRules

      • 相关规则说明详见：Frame CsvHelper

    • 在新增表的输入框内填写：TableTest，点击“新增表”后，将生成当前表的配置项。

    • 勾选字段Name的本地化设置，并选择本地化的 key 为Id；

    • 在最下面添加ID，点击 + 号；

    • 点击「一键导表」或「导出当前表」按钮。等待代码编译；

  4. Localization资源：

    • 在 Game\Localize\Text：系统会自动生成对应的本地化数据。：

      • Table：存放 表对应的本地化数据 和 Shared Data 数据
      • ZH-CN：存放中文的所有本地化表数据
      • En：存放英文的所有本地化表数据
      • 如果有其他语言，则接着生成对应本地化的文件夹

    • 打开 TableTest 的 TableEditor，可以看到生成了相关key对应的本地化文本

      • Key 的生成规则为：A.B
        • A：在 TableRules 表里，勾选了本地化的字段名；
        • B：在本地化的Key选择的字段的表实际值；为了保证 Key 唯一，通常使用 Id 作为标识。

    • 在英文栏目里，针对 Name.1 填写 Test1，针对 Name.2 填写 Test2；

  5. 表代码：

    • 在 Game\Scripts\Table：自动生成 TableTest.cs 文件（根据表名生成）

    • 打开脚本可以看到，Name 字段已经自动接入本地化获取函数。

      • 传参Key 的生成规则为：A.B.C
        • A：表名，也是本地化数据名
        • B：勾选本地化的字段名
        • C：本地化的key所选的字段的值；一般都是Id对应的行数值，为了唯一；

      using System;
using DCFrame;
using UnityEngine;
using System.Collections.Generic;

namespace Game.Table {
    public class TableTestField {
        public int Id { get; set; }
	public string Name => Localize.GetText("TableTest.Name." + Id);
	public bool IsTure { get; set; }
    }
    public class TableTest : TableBaseSingle<TableTest> {
        public TableTestField GetConfigById(int id, bool showTips = true) {
            if (keyDicId.ContainsKey(id)) {
                return keyDicId[id];
            }
            if (showTips) {	
                Debug.LogError(String.Format($"查找表：{CsvPath} 失败, id：{id}"));
            }
            return null;
        }

        public TableTest() {
            keyDicId = LoadTableDic<int, TableTestField>(x => (x.Id));
        }

        protected override string CsvPath => "Assets/Game/Table/TableTest.csv";
        private readonly Dictionary<string, TableTestField> cacheDic;
        private readonly Dictionary<int, TableTestField> keyDicId;
    }
}

      using System;
using DCFrame;
using System.Collections.Generic;

namespace Game.Table {
    public class TableEnum : TableBaseSingle<TableEnum> {
        /// <summary>
        /// 颜色标识
        /// </summary>
        public enum ColorType {
            /// <summary>
            /// 黄色
            /// </summary>
            Yellow = 1,
            /// <summary>
            /// 红色
            /// </summary>
            Red = 2
        }

        public readonly Dictionary<ColorType, Func<string>> ColorTypeDic = new() {
            [ColorType.Yellow] = () => Localize.GetText("TableEnum.ColorType.Yellow"),
            [ColorType.Red] = () => Localize.GetText("TableEnum.ColorType.Red")
        };

        /// <summary>
        /// 尺寸标识
        /// </summary>
        public enum SizeType {
            /// <summary>
            /// 大
            /// </summary>
            Big = 1,
            /// <summary>
            /// 小
            /// </summary>
            Small = 2
        }

        protected override string CsvPath => "Assets/Game/Table/TableEnum.csv";
    }
}

  6. 预制体测试：

    • 介绍：预制体上，则通过 Localize String Event 组件来填写对应 key 来处理本地化文本；

    • 在某个物体节点上添加 Text 文本，并添加 Localize String Event 组件；

      • 可在 Text 组件右上角点击三个点，选择 Localize，快速添加；

      • 也可以切换为英文环境，但由于未填写英文文本；所以会默认提示文本：

预制件工具

下面介绍 Localize String Event 组件的快捷添加方式。

  • Localization 在 Text 组件中提供了快捷添加功能，并自动绑定 Text.text 属性。

  • 工程内添加 Add Localize，从而可以自动同步绑定 String Reference 的值。

    • 在 Text 组件的文本内添加 TableString 本地化数据内的 key，如：Test1，并添加 Add Localize，系统会自动绑定 String Reference 的值为 TableString/Test1，并更改 Text 为当前本地化的中文文本：测试文本1

    • 如果需要添加其他本地化数据内的key，则需填写：LocalizeName.Key

      • LocalizeName：是本地化收集的数据名；
      • Key：当前数据内的Key
