本项目使用的 Unity 的版本是：2021.3.6f1c1

窗口说明

打开方式

  • 快捷键：Ctrl + K

  • 菜单栏：Edit -> Search All…

  • Project栏：

  • 引擎右上角的放大镜：

窗口

  1. 打开窗口的时候会显示如下内容，可以在输入框内打 ? 来寻求一系列的指引；

  2. 这里引用 Unity Manual 下的一张图；

    • A：搜索框，输入一些想要查找的资源；有关具体的搜索方法，可以参考 搜索条件；
    • B：保存搜索条件，若想保存本次搜索的条件文本，以便后续重复使用，则可以点击 User 或者 Project 旁边的保存按钮；
      • User：保存在本地；仅仅自己可见；
      • Project：保存一份资源在工程内，可以上传到Git，供项目内成员使用；
    • C：搜索结果，根据搜索框筛选出匹配中的资源；点击可以定位在工程中的位置；
    • D：分类标签，根据搜索的结果，对资源进行归类；
    • E：检视面板，允许您查看和更改项目属性、打开、重新导入或删除搜索项，或复制路径；
    • F：显示偏好，更改资源显示和排布方式；

搜索条件

  • p:：从 Project 视图内搜索；

  • = ：在 Calculator 内搜索；

  • -：排除条件搜索；

  • find:：从文件内搜索；

  • h: or t:：从 Hierarchy 内搜索；

  • #：从静态API内搜索；

  • m:：从 菜单栏 内搜索；

  • pkg:：从 资源包 内搜索；

  • at:：从 Asset Table 内搜索；

  • st:：从 String Table 内搜索；

  • set:：从 配置 内搜索；

  • store:：从 资源商店 内搜索；

  • log:：从 Los 内搜索；

  • adb:：从 Asset Database 内搜索；

  • t:：根据类型名搜索；

    • t:Prefab → 预制体
    • t:Texture → 贴图（Texture2D 等）
    • t:Material → 材质

  • l:：根据标签名搜索；

    • 在 Inspector 里给资源加 Label

  • Assets/路径关键词：在路径下搜索资源；

    • t:Prefab Assets/UI：只查 UI 目录下的预制体

  • *.XXX：扩展名搜索

    • *.png、*.fbx 、*.mat

  • size>1000：搜索大于1kb的资源

  • 组合搜索举例：

    • t:script Assets/DCFrame -Assets/DCFrame/Editor
      • 在 Assets/DCFrame 文件夹下底下搜索脚本，需排除 Assets/DCFrame/Editor 文件夹；
