Timer

  • 仓库地址：UnityTimer
  • 简要说明：在 Unity3D 中承担计时器的操作。

安装

  1. 下载压缩包：

    • 前往：Release

    • 下载最新版本的压缩包，本文下载的是 1.1.0 版本；

    • 解压压缩包，并放入工程的 Plugins 文件夹下，即可使用；

  2. Package 导入：

    • 打开 Unity -> 菜单栏 -> Window -> Package Manager

    • 选择导入 Url；

    • 输入：https://github.com/akbiggs/UnityTimer.git，然后点击Add，等待安装完成；

工程准备

  • 打开 unity.dc 工程， 创建个名为：TimerSimple 的文件夹；

  • 在文件夹下创建一个名为：TimerTest 继承至 MonoBehaviour 的脚本；

  • 在文件夹下创建一个空的预制件，上面挂载 TimerTest的脚本；

  • 打开 MainGame的脚本，改成如下代码：

    using Cysharp.Threading.Tasks;
using DCFrame;
using UnityEngine;

namespace Game {
    public class MainGame : MonoSingleton<MainGame> {
        private async UniTask Start() {
            Debug.Log("创建预制件");
            var go = await LoadAsset<GameObject>(TimerPrefabPath);
            if (go) {
                prefabTimer = Instantiate(go);
            }
        }

        public void Update() {
            if (Input.GetKeyDown(KeyCode.D)) {
                Debug.Log("销毁预制件");
                Destroy(prefabTimer);
                ReleaseAsset(TimerPrefabPath);
            }
        }

        private GameObject prefabTimer;
        private const string TimerPrefabPath = "Assets/TimerSimple/GameObject.prefab";
    }
}

  • 运行游戏，可以看到创建了个空的预制件，控制台输出 创建预制件 文本，点击键盘上的 D 按键，控制台输出 销毁预制件；此时，工程准备流程已完成；

    using UnityEngine;
using UnityTimer;
public class TimerTest : MonoBehaviour {
    void Start() {
         Debug.Log("开始计时: " + Time.time);
         timer1 = Timer.Register(2, () => { Debug.Log( "延迟2秒输出: " + Time.time); });
         Timer.Register(1, () => { 
             Debug.Log( "暂停时间: " + Time.time);
             Timer.Pause(timer1);
             Timer.Register(1, () => {
                 Debug.Log( "恢复时间: " + Time.time);
                 Timer.Resume(timer1);
             });
         });
    }

    void Update() {
        if (Input.GetKeyDown(KeyCode.A)) {
            Debug.Log("取消计时器");
            Timer.Cancel(timer1);
        }
    }

    private Timer timer1;
}
