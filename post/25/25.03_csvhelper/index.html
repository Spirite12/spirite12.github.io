介绍

CSVHelper 用于读写 CSV 文件的 .NET 库，速度极快、灵活且易于使用。

下载

  • 打开文档网站： CsvHelper Doc，点击 Download 下载按钮；

  • 跳转到下载网址后，依次选择：Versions 栏目 -> 自己所需的版本 -> 点击 Download package

  • 下载文件好后，将 .nupkg 后缀名更改成 .zip，并解压；

  • 打开文件夹下 lib -> netstandard2.1 -> 找到 CsvHelper.dll ，并放入 Unity 项目工程；

    • 请根据实际情况选择 lib 文件夹下，对应环境的文件夹内容；

public class Test : MonoBehaviour {
    void Start() {
        WriteCsvFile();
    }

    private void WriteCsvFile() {
        // 申明写入的数据
        var recordList = new List<CsvTest> {
            new() { Id = 1, Name = "测试1", IsTrue = true},
            new() { Id = 2, Name = "测试2", IsTrue = false},
        };
        var path = "Assets/Csv/Test.csv";
        using var writer = new StreamWriter(path, false, Encoding.UTF8);
        using var csv = new CsvWriter(writer, CultureInfo.InvariantCulture);
        csv.WriteRecords(recordList);
        writer.Flush();
        Debug.Log($"CSV 写入完成: {path}");
    }
	
    // 定义写入的数据结构
    private class CsvTest {
        public int Id {get; set; }
        public string Name {get; set; }
        public bool IsTrue {get; set; }
    }
}
  • StreamWriter ：第二个参数是：是否是覆盖（false）还是追加（true）形式；
  • StreamWriter ：第三个参数是：编码格式，防止乱码；

      private void ReadCsvFile() {
        using var reader = new StreamReader("Assets/Csv/Test.csv", Encoding.UTF8);
        var config = new CsvConfiguration(CultureInfo.InvariantCulture) {
            PrepareHeaderForMatch = args => args.Header.ToLower(),
        };
        using var csv = new CsvReader(reader, config);
        var records = csv.GetRecords<CsvTest>();
        foreach (var record in records) {
            Debug.Log(record.id + "-" + record.name + "-" + record.isTrue);
        }
    }

    private class CsvTest {
        public int id {get; set; }
        public string name {get; set; }
        public bool isTrue {get; set; }

      private void ReadCsvFile() {
    using var reader = new StreamReader("Assets/Csv/Test.csv", Encoding.UTF8);
    using (var csv = new CsvReader(reader, CultureInfo.InvariantCulture)) {
        csv.Context.RegisterClassMap<CsvTestMap>();
        var records = csv.GetRecords<CsvTest>();
        foreach (var record in records) {
            Debug.Log(record.Id1 + "-" + record.Name1 + "-" + record.IsTrue1);
        }
    }
}

public class CsvTest {
    public int Id1 {get; set; }
    public string Name1 {get; set; }
    public bool IsTrue1 {get; set; }
}

public sealed class CsvTestMap : ClassMap<CsvTest> {
    public CsvTestMap() {
        Map(m => m.Id1).Name("Id");
        Map(m => m.Name1).Name("Name");
        Map(m => m.IsTrue1).Name("IsTrue");
    }
}

内容记录

  • 类属性定义说明：链接

    属性 说明
    [Delimiter("|")] 指定该类在读写 CSV 时使用的分隔符；（csv文件默认以 , 分割）
    [CultureInfo(“de-DE”)] 指定使用的文化区域（德语/德国），影响数字、小数点、日期等格式。例如：德国格式小数用逗号：3,14 ，英文格式小数用点：3.14

  • 类字段属性定义字段说明：链接

    属性 说明
    [Name(“Identifier”)] 指定 CSV 文件中该列的标题名
    [Index(1)] 如果没有标题行，指定 CSV 中该列的列索引位置（从 0 开始）
    [CultureInfo(“de-DE”)] 只对该属性单独指定文化格式（覆盖类级的设置）
    [BooleanTrueValues(“yes”)] 定义布尔类型 True 在 CSV 中的表示方式；
    [BooleanTrueValues(“yes”, “1”, “true”)] ：可设置多值
    [BooleanFalseValues(“no”)] 定义布尔类型 False 在 CSV 中的表示方式；
    [BooleanFalseValues(“no”, “0”, “false”)] ：可设置多值
    [Constant(“bar”)] 如果 CSV 中没有该列，这个属性会始终被赋值为固定常量 "bar"
    [Optional] 该列是可选列，如果 CSV 文件中没有这一列，解析时不会报错
    [Ignore] 在 CSV 读写时忽略该属性，不会参与序列化（写出）或反序列化（读取）

  • CsvConfiguration 类字段说明：

    1. 基础输入输出设置
    属性 类型 说明
    Delimiter string 指定该类在读写 CSV 时使用的分隔符
    HasHeaderRecord bool 是否有表头行（第一行是列名），默认True
    TrimOptions TrimOptions 枚举 是否去掉空格。可设为 Trim、None、InsideQuotes
    IgnoreBlankLines bool 是否忽略空行，默认True
    AllowComments bool 是否允许注释行（以 Comment 开头）
    Comment char 注释标识符，默认 #。例如：# 这是注释。
    BadDataFound Action 发现格式错误时的回调，可自定义忽略或记录。
    1. 文化与格式
    属性 类型 说明
    CultureInfo CultureInfo 控制数字、小数点、日期等解析规则。默认 CultureInfo.InvariantCulture。
    DetectColumnCountChanges bool 是否检测每行列数变化，默认 false。开启后列数不一致会抛错。
    ShouldQuote Func<string, WritingContext, bool> 控制哪些值需要用引号包裹。可自定义逻辑。
    Quote char 引号字符，默认 "。
    QuoteAllFields bool 是否强制所有字段都加引号。
    Escape char 转义字符，默认 "（同 Quote）。
    Encoding Encoding 设置文件编码，例如 Encoding.UTF8。
    1. 映射与类型控制
    属性 类型 说明
    MissingFieldFound Action<ReadingContext> 当 CSV 少了某字段时的回调，默认抛异常，可设置为 null 忽略。
    HeaderValidated Action<HeaderValidatedArgs> 校验标题行时触发，可禁用验证。
    PrepareHeaderForMatch Func<string, int, string> 自定义列名匹配逻辑（如忽略大小写或空格）。
    IgnoreReferences bool 忽略引用类型属性（即导航属性，不自动展开）。
    IgnoreQuotes bool 是否忽略引号的语义。默认为 false，一般不要改。
    Mode CsvMode 枚举 控制兼容性模式，例如 CsvMode.NoEscape、CsvMode.RFC4180（默认）
    1. 性能与解析控制
    属性 类型 说明
    BufferSize int 读取缓冲区大小，默认 4096。
    NewLine string 写入时换行符，默认环境行结尾（Windows 为 \r\n）。
    ShouldSkipRecord Func<string[], ReadingContext, bool> 自定义跳过哪些行的逻辑。
    SanitizeForInjection bool 防止 Excel 公式注入（比如以 = 开头的字段），默认 false。
    DetectDelimiter bool 自动检测分隔符。
